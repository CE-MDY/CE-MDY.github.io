Aseprite, 간편 설치부터 활용까지

작성일 2021-04-05 Edited on 2021-04-04

###1. Aesprite 과의 만남

저는 Unity 2D로 게임 만들어오는 것을 선호해 왔습니다.
픽셀 도트 작업을 시작하는 사람이라면 한번 쯤은 들어봤을 법도 한 Aesprite
이 이름도 귀여운 녀석이 알고보면 어마어마하게 무시무시한 놈이라는 것입니다.

누가 너에게 최근에 가장 빡치는 일이 있냐고 물어본다면, 고개를 들어 아래를 보라

  1. Aesprite란??
    // Aesprite 이미지 파일 //
    C++ 기반으로 만든 그래픽 소프트웨어, 포토샵과 기능이 비슷하며 픽셀 도트 작업에 특화되어 있다.
    초보자도 알기 쉽게 따라하는 Aesprite 관련 링크
  1. 설치 방법
    // Aesprite 이미지 파일 //
    스팀에서 20달러에 판매하고 있지만, 다른 설치 방법이 있다면 어떻게 하겠습니까?? 당연히 찾아봐야 하지 않겠었습니까
    그것이 큰 화근이었습니다. 주말을 10시간 쏟아 부어서야 겨우 설치 할 수 있었던 Aesprite 여러분들은 고구마먹은 속답답
    느끼지 말라고 공유하게 되었습니다.

구글에 “Aesprite 다운” 이라고 치니 많은 방법들이 나와있었습니다. – 2020년 기준
관련링크 ‘이분께 우선 너무 감사하다. 10시간 삽질의 포문을 열어주셨으니’

  1. 메인, 오류 잡기

(1) 첫번째 오류
skia 컴파일

skia를 깃허브에서 git clone으로 다운 받으셨다면(zip으로 다운 받으셨다면)
이제 컴파일을 하실 텐데요. 여기서 포기는 무리무리,

의외로 해결방법은 간단했습니다
c:\deps\skia\gn\highest_version_dir.py 파일에서 수정만 하면 됩니다.

알고보니 오류는 여기에 있었습니다. print는 () 로 묶어줘야 하는데, 안묶여 있었으니 말이죠
print sorted 여기서 뜨셨던 분들이라면, print (sorted(blahblah)[-1]) 로 묶어 주세요

(2) 두번째 오류

subcommand stop 이라는 문자와 함께 갑자기 컴퓨터가 실행을 정지시키는데요
